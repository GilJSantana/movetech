# Desafio: Alerta de Gastos da Magalu Cloud
## Aluno: José Luiz Fagundes - Brasília-DF - Movetech 40+ - 2026

## Problema
Atualmente não temos uma interface acessível para que o usuário defina um limite de gastos e receba alertas sobre o consumo. A alternativa disponível exige conhecimento técnico para usar a API de consumo, tornando a solução pouco prática para a maioria dos clientes.

## Proposta de solução
Desenvolver uma solução seguindo o padrão atual da magalu cloud de uma comunição visual simples e objetiva, composta por:
- Um frontend acessível por link, onde o usuário possa definir o limite de gastos.
- Um backend que consulte a API de consumo, calcule um resumo dos gastos e compare com os limites configurados.
- Um mecanismo de notificações que avise o usuário quando o consumo alcançar determinados percentuais (por exemplo, 80% e 100% do limite).

## Critérios de aceite
- Usuário consegue definir e editar um limite via link.
- Sistema envia notificações quando os limiares são atingidos.
- É possível visualizar um resumo do consumo atual comparado ao limite.
