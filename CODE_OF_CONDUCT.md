# Código de Conduta

## Nosso compromisso

Este repositório foi criado para fins educacionais e tem como objetivo proporcionar uma experiência colaborativa utilizando Git e GitHub.

Esperamos que todos os participantes contribuam de forma respeitosa, colaborativa e profissional.

---

# Comportamento esperado

Todos os participantes devem:

- Respeitar os colegas durante toda a atividade;
- Utilizar uma comunicação educada e construtiva;
- Aceitar sugestões e feedbacks com profissionalismo;
- Colaborar para manter a organização do projeto;
- Compartilhar conhecimento sempre que possível.

---

# Durante o Code Review

Ao revisar um Pull Request:

- Faça comentários relacionados ao código;
- Explique o motivo das sugestões;
- Seja objetivo e respeitoso;
- Evite críticas pessoais;
- Valorize os pontos positivos antes de sugerir melhorias.

Exemplo:

✅ "Excelente organização do arquivo. Apenas sugiro melhorar a mensagem do commit."

Evite:

❌ "Seu código está errado."

---

# Commits

Utilize mensagens claras e objetivas.

Exemplos:

```text
docs: adiciona arquivo do aluno João

feat: cria documentação do laboratório

fix: corrige erro de formatação no README
```

Evite mensagens como:

```text
teste

alteração

aaaa

commit
```

---

# Pull Requests

Antes de abrir um Pull Request, verifique se:

- A alteração está funcionando;
- Apenas os arquivos necessários foram modificados;
- A descrição explica claramente o que foi alterado.

---

# Resolução de Conflitos

Conflitos de Merge fazem parte do desenvolvimento de software.

Caso ocorra um conflito:

- Mantenha a calma;
- Analise as alterações;
- Converse com os colegas quando necessário;
- Resolva o conflito preservando o conteúdo correto.

---

# Ambiente de Aprendizado

Este laboratório é um espaço para aprender.

Erros fazem parte do processo e serão utilizados como oportunidade de aprendizado para toda a turma.

---

# Dúvidas

Caso encontre dificuldades durante a atividade, procure o instrutor antes de realizar alterações que possam impactar o trabalho dos demais participantes.

---

Obrigado por contribuir com este laboratório!
