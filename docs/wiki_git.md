# O que é o Git?

O **Git** é um sistema de controle de versão distribuído, amplamente utilizado para gerenciar o desenvolvimento de softwares e acompanhar alterações em arquivos ao longo do tempo.

## Origem
* **Criador:** Linus Torvalds (o mesmo criador do Kernel Linux).
* **Ano:** 2005.

## Por que foi criado?
O Git foi desenvolvido para substituir o sistema proprietário (BitKeeper) que era usado no desenvolvimento do Linux. Torvalds precisava de uma ferramenta que fosse:
1. **Extremamente rápida.**
2. **Capaz de lidar com projetos gigantescos** de forma eficiente.
3. **Totalmente distribuída** (cada desenvolvedor tem uma cópia completa do histórico do projeto).

## Principais Benefícios
* **Segurança:** O histórico não pode ser alterado sem que isso seja detectado.
* **Colaboração:** Facilita o trabalho em equipe através de *branches* (ramos), permitindo que várias pessoas trabalhem no mesmo projeto sem sobrescrever o código umas das outras.
* **Offline:** Você não precisa de conexão com a internet para criar commits ou verificar o histórico; a sincronização com o servidor remoto (como GitHub ou GitLab) é feita apenas quando necessário.
