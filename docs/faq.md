# FAQ

Perguntas frequentes sobre Git e GitHub.

## O que é Git?

Sistema de controle de versão distribuído que registra o histórico de alterações de um projeto e permite o trabalho colaborativo.

## O que é GitHub?

Plataforma de hospedagem de repositórios Git na nuvem, com recursos de colaboração como Pull Requests, issues e revisão de código.

## O que é um Commit?

Registro de uma alteração no repositório. Cada commit tem um identificador único e uma mensagem que descreve o que mudou.
