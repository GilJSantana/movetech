# Guia de Contribuição

Obrigado por contribuir com o laboratório de Git e GitHub!

## Fluxo de trabalho

### 1. Clone o repositório

```bash
git clone <URL_DO_REPOSITORIO>
```

### 2. Crie uma branch

Nunca trabalhe diretamente na `main`.

Utilize o padrão:

```text
feature/seu-nome
```

Exemplo:

```text
feature/joao
```

### 3. Faça suas alterações

Implemente apenas a atividade proposta.

### 4. Realize um commit

Utilize uma mensagem clara e objetiva.

Exemplo:

```text
docs: adiciona arquivo do aluno João
```

### 5. Envie sua branch

```bash
git push origin feature/seu-nome
```

### 6. Abra um Pull Request

Descreva brevemente o que foi alterado.

### 7. Aguarde a revisão

Não realize o Merge da sua própria branch.

Após a aprovação do instrutor, atualize sua branch principal.

```bash
git checkout main
git pull
```
